# Componetns [Vocs & Custom Components]

Vocs comes with default components that can be extended or you can create and use your own.

All components available can be found in [vocs/src/components](https://github.com/wevm/vocs/blob/main/src/components.ts)

## Vocs Components

The following are components native to Vocs and that can be used throughout your docs.

## Custom Components

Create custom components for your docs.

### Extending Vocs Component

This will demonstrate how to extend an existing vocs component.

:::code-group

<div data-title="Preview">
  <MyButton.Large>Hello</MyButton.Large>
</div>

```mdx [index.mdx]
import * as MyButton from '../../../components/MyButton';

<MyButton.Large>Hello</MyButton.Large>
```

```tsx [MyButton.tsx]
import clsx from 'clsx'
import * as React from 'react'
import { Button } from 'vocs/components'

type ButtonProps = {
  children: React.ReactNode
  className?: string
  href?: string
  variant?: 'accent'
}

export function Large({ children, className, ...rest }: ButtonProps) {
  return (
    <Button className={clsx('text-xl', className)} {...rest}>
      {children}
    </Button>
  )
}
```

```css [styles.css]
/* If tailwind not used */
.vocs_Button_button.text-xl {
  font-size: var(--vocs-fontSize_32);
  height: var(--vocs-fontSize_64);
}
```
:::

  <Custom>My Custom Button</Custom>
</div>


```tsx [Custom.tsx]
import * as React from 'react'

export default function Custom({ children }: { children: React.ReactNode }) {
  return (
    <button
      style={{
        background: 'var(--vocs-color_textAccent)',
        paddingLeft: 'var(--vocs-space_16)',
        paddingRight: 'var(--vocs-space_16)',
        borderRadius: 'var(--vocs-space_4)',
        lineHeight: 'var(--vocs-space_32)',
      }}
    >
      {children}
    </button>
  )
}

```
